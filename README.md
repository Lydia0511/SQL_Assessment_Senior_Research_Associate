# Senior Research Associate SQL Assessment

## Objective

This project demonstrates SQL solutions for healthcare claims analytics.

## Question 1

Identify the top 5 providers within each specialty based on revenue earned in 2025.

### Approach

- Aggregate claims by provider
- Calculate total revenue
- Calculate total claims
- Rank providers within specialty
- Return Top 5

## Question 2

Identify open claims where both informational and true takebacks were posted for the same amount.

### Approach

- Filter takeback transactions
- Group by Claim ID and Takeback Amount
- Detect duplicate takebacks
- Return claim details

- MySQL Workbench
